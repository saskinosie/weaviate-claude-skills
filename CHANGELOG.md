## [1.0.0] - 2025-01-03

### Added
- **weaviate-local-setup** skill for Docker-based local Weaviate instances
- **weaviate-connection** skill for connecting to Weaviate (local and cloud)
- **weaviate-collection-manager** skill for schema and collection management
- **weaviate-data-ingestion** skill for uploading data with chunking strategies
- **weaviate-query-agent** skill for semantic search, RAG, and hybrid queries
- MIT License
- Contributing guidelines
- Dev container configuration for VS Code
- Comprehensive README with quick start guide
- Python requirements.txt
- Docker Compose configurations for local development
- Environment variable templates (.env.example)

### Documentation
- Complete setup instructions for Claude.ai and Claude Desktop
- Local vs Cloud workflow documentation
- Troubleshooting guides for common issues
- Example usage scenarios
- Release and GitHub setup guides

### Added

### Changed

Features that will be removed in future versions

### Removed
Features that have been removed

### Fixed
Bug fixes

### Security
Security updates or vulnerability patches
